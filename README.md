# SpringSecurity_BasicAuth

This is a Spring Boot REST API that demonstrates Basic Authentication using Spring Security with simple CRUD operations.
The application allows user registration without authentication, while secured endpoints require valid Basic Auth credentials.
User passwords are securely stored using BCrypt password encoding.
It uses Spring Data JPA for database operations and supports H2 or MySQL databases.
